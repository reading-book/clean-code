  ## Chapter 2. 의미있는 이름
- 의미가 분명한 이름 짓기
- 루프 속 i j k 사용하지 않기 : 최대한 의미를 찾을 수 있도록
- 통일성 있는 단어 사용하기
- 변수명에 타입 넣지 않기
- 구글 자바 네이밍 가이드: https://google.github.io/styleguide/javaguide.html#s5-naming

- Class Naming Guide: UpperCamelCase
